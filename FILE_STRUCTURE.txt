INSTALLATION COMMANDS
=====================

1. composer install
2. npm install
3. cp .env.example .env
4. php artisan key:generate
5. Create database: trainify
6. php artisan migrate --seed
7. php artisan storage:link
8. npm run dev (terminal 1)
9. php artisan serve (terminal 2)
10. Open: http://localhost:8000


IMPORTANT NOTES
===============

- Semua file sudah siap pakai
- Sudah ada dummy data (seeders)
- Soft delete di semua model
- Relasi lengkap dengan eager loading
- Validation di semua form
- Flash messages untuk feedback
- Role-based access dengan middleware
- Storage symlink untuk upload
- Export Excel sudah include package
